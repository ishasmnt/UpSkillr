# UpSkillr – Online Learning and Skill Development Platform

UpSkillr is a full-stack MERN-based online learning platform designed to simplify skill development through instructor-led courses. The platform allows instructors to create and manage courses, while learners can explore, enroll, and track their learning progress in real time.

This project simulates a lightweight Learning Management System (LMS) with a focus on clean architecture, role-based access, and scalable design.

---

## 📌 Project Objectives

- Build a MERN-based platform supporting Instructor and Learner roles
- Implement course creation, enrollment, and progress tracking
- Apply JWT-based authentication and authorization
- Design a modular and easy-to-understand project structure
- Follow industry-standard GitHub and development practices

---

## 🛠 Tech Stack

**Frontend**
- React
- CSS / Tailwind (optional)
- Axios

**Backend**
- Node.js
- Express.js
- MongoDB (Mongoose)

**Authentication**
- JWT (JSON Web Token)
